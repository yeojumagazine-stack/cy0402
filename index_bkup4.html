☁️ Supabase 데이터베이스 연결 미연결

여러 기기에서 데이터를 공유하려면 Supabase를 연결해야 해요.
아래 순서대로 따라하면 3분 안에 설정할 수 있어요!

🔧 설정 순서
1. supabase.com 접속 → Start your project → 구글 계정으로 가입
2. New project → 이름 입력 → 비밀번호 설정 → Region: Southeast Asia → Create
3. 왼쪽 메뉴 SQL Editor → 아래 SQL 복사 후 실행 (▶ Run)
create table if not exists app_data ( id integer primary key default 1, payload jsonb default '{}', updated_at timestamptz default now() ); insert into app_data (id,payload) values (1,'{}') on conflict do nothing; alter table app_data enable row level security; create policy "allow_all" on app_data for all using (true) with check (true); alter publication supabase_realtime add table app_data; 4. ⚙️ Project Settings → API → URL과 anon key 복사해서 아래에 입력
